# Shashki.com — онлайн-платформа для игры в шашки

Шахматный Chess.com, но для шашек. Полноценная веб-платформа с онлайн-игрой, ИИ-коучем и анализом партий.

## Что сделано

- **Три варианта правил** — русские, международные, английские шашки с полным движком (взятие, дамки, обязательные взятия)
- **Игра против бота** с тремя уровнями сложности (лёгкий / средний / сложный)
- **Онлайн-игра** — matchmaking, приватные комнаты, таймер с контролем времени (3+2, 5+0, 10+0)
- **ИИ-коуч** на Claude Haiku — даёт подсказки и отвечает на вопросы прямо во время партии, знает текущую позицию и доступные ходы
- **Разбор партий** — пошаговый реплей с AI-анализом каждого хода
- **Профиль и рейтинг** — ELO-система, история партий, таблица лидеров
- **Аутентификация** — регистрация / вход через Supabase Auth

## Для кого

Для игроков в шашки любого уровня — от новичков, которым нужен коуч, до тех, кто хочет играть онлайн с друзьями и соперниками.

## Почему это ценно

Шашки — игра с многомиллионной аудиторией, но достойной современной платформы нет. Shashki.com закрывает этот пробел: это первая платформа с ИИ-коучем, который объясняет ходы на русском языке и адаптируется под уровень игрока.

## Стек

- **Frontend**: React 19 + TypeScript + Vite
- **State**: Zustand
- **Backend / Auth / Realtime**: Supabase
- **AI**: Anthropic Claude API (claude-haiku-4-5)
- **Деплой**: Vercel

## Запуск локально

```bash
npm install
# Создай .env.local с VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY, VITE_ANTHROPIC_API_KEY
npm run dev
```
